The free options (genuinely worth trying first)

  • Pricefy free tier — up to 50 SKUs / 5 competitors, no credit card. Great starting point.
  • changedetection.io — open-source, self-hosted, very capable if you're comfortable setting it up.
  • Google Alerts / manual checks — free but doesn't scale and misses fast moves.

If those fit, use them. The gap they leave: the free SaaS tiers cap you, and the open-source route takes real setup time.
